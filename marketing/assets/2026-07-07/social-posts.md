## LinkedIn Draft

KISKIR added a public project page for Life in Weeks, a static browser tool that turns timelines and public biographies into a 100-year week grid.

The useful detail is not only the grid. Exact events can pin to week cells, while year-only, month-only, or age-level public facts stay visibly approximate instead of being forced into false precision.

The project also includes a reusable public-figure profile workflow for coding agents, plus exportable visual output for sharing a timeline outside the app.

Project page:
https://kiskir.dev/projects/life-in-weeks/

Public app:
https://kiskir.dev/life/

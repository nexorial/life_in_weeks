# AGENTS.md - Life In Weeks

## Role

This project owns the Life in Weeks web page served at:

`https://kiskir.dev/life`

## Rules

- Keep the app static, local-first, and easy to deploy.
- Do not add runtime dependencies unless the feature cannot be implemented cleanly with platform APIs.
- Verify the visual surface in a browser before claiming completion.
- Keep the share/export output useful as a saved image, not just a screenshot of controls.
- When publishing to `kiskir.dev/life`, update the `official-website` route copy as part of the same release.
